# PRODIGY PRO - استثمر في الجيل القادم من العمالقة

موقع تعريفي احترافي لمشروع **PRODIGY PRO**، منصة استثمارية ديمقراطية توفر فرص الاستثمار المبكر في الشركات الناشئة المبتكرة.

## 🌟 نبذة عن المشروع

**PRODIGY PRO** هو بوابة ديمقراطية للاستثمار في الشركات الناشئة المبتكرة. نحن نكسر احتكار رأس المال الاستثماري ونوفر لكل مستثمر فرصة الاستثمار المبكر في الشركات التي ستغير العالم.

### المشكلة التي نحلها:
- **الوصول الحصري**: الفرص الذهبية محصورة على المستثمرين الأثرياء فقط
- **فجوة الثروة**: فقط رؤوس الأموال الضخمة تجني الأرباح من النمو المبكر
- **عدم وجود جسور**: لا توجد طريقة آمنة لربط السيولة الرقمية بالمشاريع الحقيقية

### الحل الذي نقدمه:
- **مفتاح الوصول**: تذكرة دخول مبكرة لمشاريع تقنية وحقيقية
- **ربط الرقمي بالحقيقي**: تحويل العملات الرقمية إلى استثمار حقيقي
- **نجاح ديمقراطي**: كل مستثمر يمكنه أن يكون مستثمراً مبكراً

## 🎨 التصميم والهوية البصرية

الموقع مصمم بأسلوب **Dark Luxury Tech** يعكس الفخامة والابتكار:

- **الألوان الأساسية**: أسود عميق (#0A0F0D) مع أخضر نيوني (#00FF88)
- **التأثيرات**: توهج (Glow) وزجاج (Glassmorphism)
- **الخطوط**: Playfair Display للعناوين، Poppins للنصوص
- **التفاعلات**: انتقالات سلسة وتأثيرات Hover جذابة

## 📱 الأقسام الرئيسية

1. **البطولة (Hero Section)**: مقدمة جذابة مع معلومات المؤسسين
2. **المشكلة**: عرض التحديات في الاستثمار التقليدي
3. **الحل**: الميزات الفريدة لـ PRODIGY PRO
4. **كيف تربح**: محركات الثروة الثلاث
5. **دورة القيمة**: رحلة من الاكتشاف إلى العائد
6. **اقتصاد الرموز**: تفاصيل العرض والتوزيع
7. **الفريق**: معلومات المؤسسين والخبرات
8. **الاتصال**: نموذج اتصال ومعلومات التواصل المباشرة
9. **الـ Footer**: روابط وتفاصيل إضافية

## 🛠️ التقنيات المستخدمة

## 🚀 البدء السريع

### المتطلبات:
- Node.js 18+
- pnpm أو npm

### التثبيت والتشغيل:

```bash
# استنساخ المستودع
git clone https://github.com/PRODIGY406/prodigy-pro-site.git
cd prodigy-pro-site

# تثبيت المتطلبات
pnpm install

# تشغيل خادم التطوير
pnpm dev

# سيكون الموقع متاحاً على http://localhost:3000
```

### البناء للإنتاج:

```bash
# بناء الموقع
pnpm build

# معاينة النسخة المبنية
pnpm preview
```
